<Update label="2024-10-15" description="v1.2.0" tags={["feature", "bugfix"]}>

- Added real-time caregiver location tracking using geolocation APIs.
- Introduced patient vital monitoring integration with wearable devices.
- New dashboard widget for upcoming appointment summaries.

## Bug Fixes

- Fixed scheduling conflicts that caused overlapping caregiver shifts.
- Resolved mobile app crashes during offline mode sync.
- Corrected notification delays for urgent care requests.

## Improvements

- Enhanced search functionality for patient records with fuzzy matching.

</Update>

## Upgrade Instructions

Follow these steps to upgrade PRATHAM RAVAL smoothly. Test in staging before production.

<Steps>
  <Step title="Backup Data" icon="database">
    Export your patient and scheduling data.
  </Step>
  <Step title="Update Package" icon="download">
    
````bash
npm install pratham-raval@latest
````

  </Step>
  <Step title="Migrate Config" icon="settings">

    <Tabs>
      <Tab title="v1.0 to v1.1" icon="arrow-right">
        Update your config to use the new JWT endpoints.
        
````javascript
// Old
const auth = { apiKey: 'YOUR_API_KEY' };

// New
const auth = { 
  token: 'YOUR_JWT_TOKEN',
  endpoint: 'https://api.example.com/auth/v2'
};
````
      </Tab>
      <Tab title="v1.1 to v1.2" icon="check-circle">
        Enable geolocation in settings.
        
````json
{
  "features": {
    "geotracking": true,
    "wearables": "fitbit"
  }
}
````
      </Tab>
    </Tabs>

  </Step>
  <Step title="Verify" icon="shield">
    Run tests and check caregiver dashboard.
  </Step>
</Steps>
