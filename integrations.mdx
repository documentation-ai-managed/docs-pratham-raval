## Sync with Google Calendar

Set up two-way sync to manage caregiver schedules effortlessly.

<Steps>
  <Step title="Enable Integration" icon="settings">
    Navigate to Integrations in your dashboard. Select Google Calendar and authorize with your account.
  </Step>
  <Step title="Map Events" icon="map">
    Create rules to map care visits to calendar events. Use patient names in titles for easy recognition.
  </Step>
  <Step title="Test Sync">
````bash
# Verify sync by creating a test visit
curl -X POST https://api.example.com/visits \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{"patientId": "123", "start": "2024-10-20T10:00:00Z"}'
````
    Check your calendar for the new event.
  </Step>
</Steps>

## Webhooks for Notifications

Set up webhooks to trigger actions on events like visit completions.

<CodeGroup tabs="Dashboard Setup,cURL">
````javascript
// Example webhook payload handler
app.post('/webhook', (req, res) => {
  const event = req.body;
  if (event.type === 'visit.completed') {
    // Notify caregiver via Slack
    console.log(`Visit ${event.data.visitId} finished`);
  }
  res.status(200).send('OK');
});
````
````bash
# Register webhook in dashboard
curl -X POST https://api.example.com/webhooks \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{"url": "https://your-webhook-url.com/home-care", "events": ["visit.completed"]}'
````
</CodeGroup>

## Advanced API Integrations

Access full API for custom workflows.

<ParamField path="visitId" param-type="string" required="true">
  Unique visit identifier.
</ParamField>

<ParamField header="Authorization" param-type="string" required="true">
  Use `Bearer YOUR_API_KEY`.
</ParamField>

<Request tabs="Create Visit,Get Visits">
````javascript
const response = await fetch('https://api.example.com/visits', {
  method: 'POST',
  headers: { 'Authorization': 'Bearer YOUR_API_KEY', 'Content-Type': 'application/json' },
  body: JSON.stringify({
    patientId: '123',
    caregiverId: '456',
    startTime: '2024-10-20T10:00:00Z'
  })
});
````
````javascript
const visits = await fetch('https://api.example.com/visits?patientId=123', {
  headers: { 'Authorization': 'Bearer YOUR_API_KEY' }
});
````
</Request>

<Response tabs="200,400">
````json
{
  "id": "visit-789",
  "status": "scheduled",
  "patient": { "name": "John Doe" }

````
